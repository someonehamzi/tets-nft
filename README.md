# Signal Entities — Cinematic Pre-Mint Prototype

This version intentionally avoids the normal NFT landing-page layout.
It is a single interactive signal-hunting interface:

- manual frequency tuner
- unknown entity reveal
- procedural audio using the Web Audio API
- waveform canvas animation
- archive-task flow for X link + EVM address
- corrupted signal submission state

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

Audio starts only after clicking **Enable Audio** because browsers block autoplay audio.
The images are local demo assets from the examples you provided.
